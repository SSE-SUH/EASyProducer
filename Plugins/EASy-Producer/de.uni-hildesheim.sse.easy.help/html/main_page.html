EASy-Producer is a Software Product Line Engineering (SPLE) tool which facilities the most recent trends and concepts in SPLE, such as large-scale Multi-Software Product Lines (MSPL), product line hierarchies, and staged configuration and instantiation.

This documentation includes the following guides and specifications:

  • EASy-Producer User Guide: Introduction to the concepts of EASy-Producer and how to use this tool.
  • EASy-Producer Developers Guide: Introduction to the concepts of extension in EASy-Producer and how to extend this tools.
  • IVML Language Specification: Introduction to and a complete specification of the IVML variability modeling language used in EASy-Producer.
  • VIL Language Specification: Introduction to and a complete specification of the VIL variability implementation language used in EASy-Producer.
