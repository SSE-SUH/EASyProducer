Documentation

Further information on how to install and use EASy can be found in the EASy Producer User Guide.

For detailed information on how to use IVML, the variability modeling language of EASy-Producer, read the release version of the IVML Language Specification (or the updated preview specification of IVML for the next release).

For detailed information on how to use VIL, the variability instantiation language of EASy-Producer, read the VIL Language Specification (or the updated preview specification of VIL for the next release).

EASy also provides extension points for the integration of custom reasoners and instantiators. A step-by-step instruction on how to extend EASy with your custom functionalities can be found in the EASy Producer Developers Guide.
